## 📤 How to Check a New Loan Application

### What You Need:
- A PDF file of the CTOS report (the document about the company)
- Customer's name
- How much money they want to borrow

### Step-by-Step Instructions:

#### STEP 1: Click "New Application"

Look at the left side menu and click **"➕ New Application"**

#### STEP 2: Fill in the Form

You'll see a form with empty boxes. Fill them in:

**Box 1 — Customer Name:**
```
Type: ABC Company Sdn Bhd
```
(Or whoever the customer is)

**Box 2 — Requested Credit Limit:**  
```
Type: 150000
```
(This means RM 150,000. Just type the numbers, no "RM" or commas)

**Box 3 — Salesman:**
```
Type: John Tan
```
(Whoever is handling this customer)

**Box 4 — Notes (Optional):**
```
Type: Returning customer, urgent request
```
(You can skip this if you want)

#### STEP 3: Upload the PDF File

Now you need to upload the CTOS report (PDF file):

**Option A — Drag and Drop:**
1. Find the PDF file on your computer
2. Click and HOLD the file
3. Drag it to the upload box
4. Let go!

**Option B — Browse:**
1. Click the **"Choose File"** button
2. A window opens showing your computer files
3. Find the PDF file
4. Click it once
5. Click **"Open"**

You should see the file name appear in the upload box! ✅

#### STEP 4: Start the Magic! 🪄

Click the big button that says:
```
🤖 Start Agent Processing
```

#### STEP 5: Watch the Robot Work!

Now the fun part! You'll see messages appearing:

```
🔵 Uploading PDF...
🔵 Reading the document...
🔵 Finding important information...
🔵 Calculating risk score...
🔵 Making decision...
🟢 Done!
```

**This takes about 20-40 seconds.** You can watch it happen in real-time!

> 💡 **Important:** Don't close the window or press the back button while this is happening!

#### STEP 6: See the Results!

After the robot finishes, you'll automatically go to a new page showing:

- ✅ All the information the robot found
- 📊 Risk score (like a grade: Good/Medium/Bad)
- 💬 Explanation in Bahasa Malaysia
- 💬 Same explanation in English
- 👍 Recommendation: Should approve or reject?

## 👀 Understanding the Results

After the robot finishes, you'll see a lot of information. Don't worry! Let me explain:

### Part 1: Company Information Found

The robot shows you what it found in the PDF:

```
Company Name: ABC Company Sdn Bhd ✅ (99% sure this is correct)
Registration: 123456-K ✅ (98% sure)
Address: No 123, Jalan XYZ... ✅ (95% sure)
Capital: RM 500,000 ✅ (99% sure)
```

The ✅ and percentage means how confident the robot is.
- ✅ 99% = Very confident, probably correct!
- ⚠️ 85% = Not very confident, maybe check this manually

### Part 2: Risk Score

You'll see a colorful bar or number:

```
Risk Score: 75% 🟢 LOW RISK
```

**What does this mean?**
- **Above 70% = GREEN = Low Risk** → Safe to approve! ✅
- **40-70% = YELLOW = Medium Risk** → Need to check carefully ⚠️
- **Below 40% = RED = High Risk** → Better to reject! ❌

### Part 3: Recommendation

The robot tells you what to do:

**If LOW RISK (GREEN):**
```
✅ RECOMMENDED: APPROVE
Suggested limit: RM 150,000
Reason: Good payment history, no problems
```

**If MEDIUM RISK (YELLOW):**
```
⚠️ RECOMMENDED: REVIEW CAREFULLY
Suggested limit: RM 50,000 (lower than requested)
Reason: Some late payments, need to check more
```

**If HIGH RISK (RED):**
```
❌ RECOMMENDED: REJECT
Reason: Too many problems, high risk
```

### Part 4: Explanation (2 Languages)

You'll see the same explanation written twice:

**Malay Version:**
```
Permohonan ini DILULUSKAN dengan had RM 150,000 kerana:
- Skor CTOS yang baik
- Sejarah bayaran yang konsisten
- Tiada masalah undang-undang
```

**English Version:**
```
This application is APPROVED with limit RM 150,000 because:
- Good CTOS score
- Consistent payment history
- No legal problems
```

Both say the same thing, just in different languages!

## ✅ How to Make Your Final Decision

After seeing the robot's recommendation, YOU need to make the final decision.

### If Robot Says "APPROVE" (Green) ✅

1. Click **"Decision & Approval"** in the left menu
2. You'll see the same application
3. Read everything one more time
4. If you agree, click the green button: **"✅ Confirm Approval"**
5. Done! The customer gets approved!

### If Robot Says "REJECT" (Red) ❌

1. Click **"Decision & Approval"** in the left menu
2. You'll see the application
3. Read the reasons why it's rejected
4. If you agree, click the red button: **"❌ Confirm Rejection"**
5. Done! The customer is rejected

### If Robot Says "REVIEW" (Yellow) ⚠️

This means the robot is not 100% sure. YOU need to decide!

1. Click **"Decision & Approval"** in the left menu
2. Look at all the information carefully
3. Read why the robot is not sure
4. Check the original PDF if needed
5. Make your own decision:
   - Click **"✅ Approve"** if you think it's safe
   - Click **"❌ Reject"** if you think it's too risky
6. Type WHY you made this decision in the notes box
7. Click **"Confirm"**

## 🆘 Help! Something Went Wrong!

### Problem 1: "Can't Upload PDF"

**Why it happens:**
- File is too big (more than 10MB)
- File is not a PDF (maybe it's a Word document or image?)
- Internet connection is slow

**How to fix:**
1. Make sure the file ends with ".pdf"
2. Check file size (right-click → Properties → Size)
3. If too big, try to compress it first
4. Check your internet connection

---

### Problem 2: "Robot is Taking Too Long"

**Why it happens:**
- PDF is very long (many pages)
- Internet is slow
- Too many people using at the same time

**How to fix:**
1. Wait 1-2 minutes (don't click anything!)
2. If nothing happens after 2 minutes, refresh the page (press F5)
3. Go to "Applications Queue" to check if it finished
4. Try again if it failed

---

### Problem 3: "Wrong Information Found"

**Why it happens:**
- PDF was scanned (photo) instead of digital text
- PDF quality is bad
- Information in PDF is unclear

**How to fix:**
1. Look at the confidence percentage
2. If below 90%, it might be wrong
3. Click the **"Edit"** button next to the wrong information
4. Type the correct information
5. Click **"Save"**
6. The robot will recalculate everything with the correct info!

---

### Problem 4: "I Don't Agree with the Robot"

**Why it happens:**
- You have information the robot doesn't know
- Special circumstances
- Customer is a VIP or returning customer

**How to fix:**
1. You are the boss! You can override the robot's decision
2. Go to **"Decision & Approval"**
3. Make your own decision
4. Write WHY you made this decision in the notes
5. Click **"Confirm"**
6. Your decision will be saved and everyone can see why you decided differently

## 📊 Understanding the Numbers

### What is "Risk Score"?

Think of it like a school grade:

- **90-100% = A** → Excellent! Very safe! ✅
- **70-89% = B** → Good! Usually safe ✅
- **50-69% = C** → Average, need to check ⚠️
- **40-49% = D** → Not good, probably reject ⚠️
- **Below 40% = F** → Fail! Too risky! ❌

### What is "Credit Limit"?

This is how much money the customer can borrow.

Example:
- Customer asks for: **RM 200,000**
- Robot recommends: **RM 150,000**
- Why lower? Because there's some risk, so give less money to be safe!

You can change this number if you want! Just type a different number.

### What is "CTOS Score"?

CTOS is like a report card for companies:
- **800-900** = A+ (Excellent!)
- **700-799** = B (Good)
- **600-699** = C (OK)
- **Below 600** = D/F (Not good)

Higher number = better company = lower risk!
